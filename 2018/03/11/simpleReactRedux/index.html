简化版ReactRedux

Posted on 2018-03-11

实现了一个最简单的react-redux，实现了核心组件Provider及核心函数connect，没做错误处理，并简化了Connect组件，去掉了非必要属性，并且无法嵌套，只能单一使用，但是该简单实现，基本保留了源码的设计思想，如selector, factory等

React & PropTypes

首先导入依赖，React和PropTypes

Provider

Provider组件，提供了最核心的功能，将store挂载到context上，传入子组件，并且render返回值为props.children

框架源码及分析见Provider.js

connect

connect函数，与源码实现方式不太相同，这里简化了逻辑

框架源码及分析见connect.js

源码中是函数式编程的写法，connect函数由是connectAdvanced.js生成

源码中对于mapStateToProps, mapDispatchToProps, mergeProps还需要用factory，wrap等函数进行处理，这里就简化，直接自己定义

给mapStateToProps,mapDispatchToProps函数定义dependsOnOwnProps，即是否依赖ownProps，该值在sourceSelector处理state和props时会用到
